Приклад транзакції SQL

Приклад SQL-транзакції для операції бронювання:

BEGIN;
-- Початок атомарної операції
UPDATE Seats
SET status = 'PENDING',
    booked_by_transaction_id = 'T'
WHERE seat_id = 'F12A' AND status = 'AVAILABLE'
FOR UPDATE;
-- Блокує вибране місце
INSERT INTO Bookings (booking_id, user_id, seat_id, booking_date, status)
VALUES ('B', 'U', 'F12A', NOW(), 'CONFIRMED');
-- Запис нового бронювання
UPDATE UserAccounts
SET balance = balance - 150.00
WHERE user_id = 'U' AND balance >= 150.00;
-- Списання коштів
COMMIT;
-- Фіксація змін

Блокування місця

Блокує запис про місце для уникнення конкурентних змін до завершення транзакції.

Запис бронювання

Додає новий запис до таблиці Bookings із деталями бронювання.

Обробка платежу

Оновлює баланс користувача із перевіркою, чи вистачає коштів.

Фіксація транзакції

Зберігає всі зміни, якщо немає помилок. Інакше — відкат.
